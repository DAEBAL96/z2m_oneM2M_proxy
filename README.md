# z2m_oneM2M_proxy application

## 개요 및 목적
z2m interface를 이용하여 AE에서 다양한 zigbee GW & device 들의 통합 객체를 이루고 사용 환경에 따라 HTTP & MQTT protocol 지원을 가능케 한다.

## 환경 및 구성요소 setup

- Node.js
- z2m(zigbee2mqtt)
- mosquitto(mqtt broker)
